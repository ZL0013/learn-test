    vuePress-theme-reco    2021 - 2024

    设置语言

    小邪 2024/1/1

    默认语言为英文
    所以如果设置文章信息的话,日期不是国内常见格式
    所以需要设置语言为中文

    修改 config.js





     
     
     
     
     
     


    module.exports = {
    title: 'Hello Vuepress',
    description: 'Hello Vuepress',
    theme: 'reco',
    // 这里添加,内容一致,不能修改
    locales: {
        '/': {
            lang: 'zh-CN'
        }
    },
}
